# Inicializar Git

En este nivel, inicializaremos un repositorio Git para rastrear nuestros cambios.

## Pasos

### 1. Inicializar Git

Ejecuta el siguiente comando en tu terminal:

```bash
git init
```

### 2. Crear .gitignore

Crea un archivo `.gitignore` para excluir `node_modules`:

```bash
echo "node_modules" > .gitignore
```
